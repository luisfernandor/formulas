Area de las figuras [Formulas]

  • Cuadrado

    La formula para resolver un cuadrado es:

    Lado * Lado

  • Rectangulo

    La formula para resolver un rectangulo es:

    Base * Altura

  • Triangulo

    La formula para resolver un triangulo es:

    Base * Altura / 2

  • Rombo

    La formula para resolver un rombo es:

    Diagonal mayor * Diagonal menor / 2

  • Romboide

    La formula para resolver un romboide es:

    Base * altura

  • Trapecio

    La formula para resolver un trapecio es:

    Altura * (Base mayor + Base menor) / 2
